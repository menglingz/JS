### 优化总结

#### 1.定时器优化

- 存在问题：开辟的定时器变量会占内存
- 优化步骤：将定时器变量作为元素的属性，这样既可以不用在内存中额外开辟一个定时器变量内存

#### 2.比较运算符使用选择

- 判断是其是否为`null`，使用双等（`jQuery`源码）

  ```js
  element == null
  // 相当于 element === null || element === undefined
  ```

- 其余比较操作用三等

#### 3.null和undefined

- `null`表示被分配了一块空内存，生成一个哈希地址
- `undefined`表示声明的变量为赋值，参与运算的时候，表达式的结果是`NaN`
